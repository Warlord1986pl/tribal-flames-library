# MTG Content Library

This repository is a structured library for Magic: The Gathering content including MTGO leagues, Beyond MTG decks, strategy articles, sideboard guides, and metagame analysis tools.

## Sections

- **MTGO Leagues:** Organized by format and league number, each league folder contains decklist, notes, and results.
- **Beyond MTG:** Long-term deck projects, each deck has a documentation file and decklist.
- **Articles:** Strategy articles and sideboard guides.
- **Metagame Tool:** Documentation and links to your metagame analysis tool.

Links to YouTube and Metafy content can be added here.
