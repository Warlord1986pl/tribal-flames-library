# TAG DEFINITIONS

Mana screw  
Brak wystarczającej liczby landów lub brak odpowiedniego koloru many przez kilka pierwszych tur, co uniemożliwia normalne rozwinięcie gry.

Flood  
Nadmierne dobieranie landów w mid/late game prowadzące do utraty tempa lub braku presji.

Execution mistake  
Błąd techniczny w prowadzeniu gry (zła sekwencja, złe zagranie, nieoptymalne użycie removalu, brak zagrania wokół znanej karty).

Mulligan issue  
Zbyt agresywny mulligan lub keep ryzykownej ręki, który znacząco wpłynął na wynik gry.

Sideboard Underperformed  
Plan sideboardowy nie poprawił matchupu lub karty sideboardowe miały niski impact.

Sideboard Overperformed  
Karty sideboardowe miały kluczowy wpływ na wygraną lub wyraźnie poprawiły matchup.

Variance heavy  
Gra w dużym stopniu rozstrzygnięta przez losowość (topdeck, brick przeciwnika, sekwencja drawów), a nie decyzje.

Deck issue  
Strukturalny problem konstrukcji decku (np. zbyt mało threatów, konflikt kart, brak odpowiedzi na typowy archetyp).

Opponent high roll  
Przeciwnik miał bardzo silny start lub idealną sekwencję zagrywek znacznie powyżej średniej.

Clean win  
Wygrana bez istotnych błędów i bez wyraźnego wpływu wariancji – plan decku zrealizowany zgodnie z założeniem.
